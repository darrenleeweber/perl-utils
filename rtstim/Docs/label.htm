Event Labels
Label Description
Event The event number given in the list above.
time The time of the event in milliseconds. The origin is set relative to the first event and any event that zeros the clock..
item The specific sound or image to be used. The item number is given by the order of the .wav or .bmp file as given in sound_list.txt or image_list.txt (starts at 0).  
pport_value The value to write to the parallel port in hex.
sync_interrupt_num The number of sync events on the parralel port before resuming the scheduling of the next events.
zero_clock Set to 1 if you want to zero the clock.
start_wav Set to 1 to start the .wav file playback or 0 to stop.
bit_blitx      bit_blt_y Set the x (horizontal) and y (vertical) positions of the bit blit rectangle. The origin is the upper left corner of the screen.
speaker_on Set to 1 to turn sound on for a "speaker" event and 0 to turn it off.
speaker_freq The speaker frequency for a "speaker" event
volume_ctrl Designate which stereo channel will have its volume set.
volume_level The  value of volume for the channel.
start_movie Set to 1 to start a movie and 0 to end the movies.
movie_rate Set the movie rate.
movie_start_index Similar to item, this label gives the first frame of the movie as given by the order of the .bmp files in image_list.txt.
movie_end_index Similar to item, this label gives the last frame of the movie as given by the order of the .bmp files in image_list.txt.
port_num The port num 1 or 2 on the DIO-24 port
port_value The value to write to the DIO-24 port.
