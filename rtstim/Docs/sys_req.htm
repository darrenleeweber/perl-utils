System Requirements

 
RTStim 2.0 has specific software and hardware requirements. Those requirements are given here.
 

1. Software Requirements

  • Redhat 9
  • RTAI 24.1.11 RTHAL-enabled Linux 2.4.20 kernel
  • RTStim

    •  2. Hardware Requirements

  • Uniprocessor x86 PC
  • ATI Radeon 7500 or 9000 AGP graphics card with DVI output
  • PCI graphics card supported by Linux 2.4.20 kernel
  • RME Digi96/8 PST or PAD pro audio card
  • Measurement Computing PCI DIO-24 card
  • Linux compatible monitor
  • LCD or DPI projector with DVI input
    • The ATI Radeon AGP graphics card is connected to a DVI projector for stimulus presentation. The PCI graphics card is connected to a monitor for normal X windows use and to display the RTStim GUI for monitoring of the experiment as it proceeds.

    We have run RTStim on six different motherboards. On the SuperMicro P6SBA motherboard some special tweaks were neccessary. If you have this motherboard send us an email and we will walk you through the tweaking.
